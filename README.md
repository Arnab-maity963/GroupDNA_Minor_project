# GroupDNA_Minor_project

This project analyzes a WhatsApp group chat dataset using Python. It performs data cleaning, message parsing, user activity analysis, time-based analysis, and visualization using NumPy.

#  WhatsApp Group Chat Analysis Project

##  Project Overview

This project analyzes a WhatsApp group chat dataset using Python.  
It converts raw chat data into meaningful insights like user activity, time patterns, word frequency, and behavioral analysis.

The goal is to perform real-world **data analysis using Python and NumPy**.

##  Dataset Description

The dataset used in this project is a WhatsApp chat export file named:

###  Format of Data:
DD/MM/YY, HH:MM - Sender Name: Message

###  Dataset Contains:

-  Timestamp of messages
-  Sender name
-  Chat messages
-  Media messages (<Media omitted>)
-  Deleted messages
-  System messages

###  Usage of Dataset:

This dataset is used to analyze:

- User activity patterns
- Most active users
- Time-based behavior (hour/day)
- Word frequency analysis
- Group engagement insights

---

##  Features

-  Chat parsing and cleaning
-  User activity analysis
-  Message count per user
-  Most active day detection
-  Most active hour analysis
-  NumPy heatmap (6×24 matrix)
-  Night activity detection
-  Word frequency analysis
-  User personality classification
-  Final report generation
